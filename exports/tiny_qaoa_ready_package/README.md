# Tiny QAOA-Ready QUBO Package\n\nThis package contains a tiny 6-variable QUBO suitable for Qiskit Optimization and QAOA-oriented toy experiments.\n\nThis package is intentionally tiny. It is not the sample_4x4 QUBO.\n\n## Files\n\n- qubo_coefficients.csv: upper-triangular QUBO coefficients.\n- qiskit_qubo.json: Qiskit-friendly QUBO representation.\n- bruteforce_energy_table.csv: exhaustive energy table over all 64 assignments.\n- package_metadata.json: package metadata and known optimum.\n\n## Known optimum\n\nBest bitstring: 101001\nBest energy: 4.0\nBest is feasible: True\n\nThis step does not run Qiskit, QAOA, quantum hardware, or a quantum simulator.\n
