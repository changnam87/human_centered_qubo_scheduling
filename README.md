# Human-Centered Quantum-Ready QUBO Scheduling

This repository supports a research project on human-centered industrial scheduling formulated as a quantum-ready QUBO/Ising optimization model.

## Research Positioning

This project is not designed to claim quantum advantage. Instead, it develops a human-centered industrial scheduling formulation that can be represented as a QUBO/Ising model and benchmarked using classical, heuristic, simulated quantum, and quantum-hybrid solvers.

## Target Journal

Computers & Industrial Engineering

## Core Problem Elements

- Task assignment
- Human/robot/machine resource allocation
- Processing time
- Precedence constraints
- Workload
- Ergonomic risk
- Skill compatibility
- Safety penalty
- Robot idle time
- Schedule feasibility

## Initial Stage

The first stage uses a small toy instance for brute-force validation of the QUBO penalty structure.

## Long-Term Extensions

- JSPLib / OR-Library / PSPLIB / FJSPLib benchmark augmentation
- Synthetic human-centered attribute generation
- MILP / CP-SAT baseline
- Simulated annealing
- QAOA simulator
- D-Wave Ocean / hybrid solver
